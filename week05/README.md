# 5주차 과제: GPT 모델 forward() 함수 구현하기

이번 주 과제는 GPT2 모델의 `forward()` 함수를 구현해보고 각기 다른 파라미터 사이즈를 가진 GPT 모델들을 생성해보는 것입니다.

각자 본인 이름 폴더 안에 있는 `ipynb` 파일을 열고, 노트북에 표시된 `TODO` 부분을 채워주세요.

## 진행 방법

1. 본인 이름 폴더로 이동합니다.
2. `ipynb` 파일을 엽니다.
3. 위에서부터 순서대로 실행하며 `TODO` 주석이 있는 코드를 완성합니다.

## 제출 위치

완성한 노트북은 본인 이름 폴더 안에 그대로 제출하면 됩니다.

```text
week04/{본인 이름}/*.ipynb
```

## 제출 체크리스트

- 모든 `TODO` 코드를 채웠는지 확인
- 노트북을 처음부터 끝까지 다시 실행했을 때 에러가 없는지 확인
